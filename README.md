# Travel Advisor

A Symfony 4 project which purpose is to sort a bucket of travel cards
Which are initially unordered and their start and end is unknown

## Getting Started

To be able to run the project you just need to run composer install. And then after that to run bin/console server:start in order to use symfony web server
The API can be reached 
And Also to have postman, since the project is 
really simple API, which at the moment has only POST endpoints but, application allows GET, PUT, PATCH, POST and DELETE.
